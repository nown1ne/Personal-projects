## Pokedex (https://github.com/nown1ne/pokedex)
A pokemon index built with https://pokeapi.co/ and Svelte Framework  

To run this locally
- clone the repo
- in the cloned folder run ``` npm i ```
- then run ```npm run dev -- --open ```
